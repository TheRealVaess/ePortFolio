Site pour les licences professionnelles de l'IUT de Montpellier

❮ ❯

Ce projet consistait à faire un site web dynamique pour les élèves des licences professionnelles de notre IUT. Ce projet a été fait en groupe de quatre durant le second semestre de mon DUT d'informatique.

Le site n'a pas été utilisé par l'IUT car ce projet devait être fait par tous les groupes et servait donc d'entraînement pour la gestion de projet avec des méthodes agiles.

Pour le créer, nous avons utilisé le CMS Wordpress qui est l'un des plus connu et qui nous permettrait donc d'exploiter le grand nombre de plugin disponibles.

Site Web programmé en HTML, CSS, et JavaScript.
