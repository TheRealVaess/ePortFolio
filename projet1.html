RETOUR ACCUEIL

Voici le projet de site web que j'ai réalisé en collaboration avec Anthony Ilhat au cours de mon premier semestre de DUT informatique : MilkyWay.
Nous avons réalisé ce projet pendant la moitié du 1er semestre et nous l'avons ensuite présenté en fin de semestre.

Le principe était que nous devions réaliser un site web sur le thème de notre choix. Nous avons donc choisi un site web vendant des planètes, étoiles, astéroïdes, etc...
Sur la photo ci-contre, vous voyez la page d'accueil du site.

Ce site web a été programmé en HTML, CSS, et JavaScript, sans Framework (comme Bootstrap), et est entièrement Responsive.

Site Web entièrement programmé en HTML et CSS.
